## Aston - Config Server

Часть микросервисного RESTful Spring Boot приложения.

Централизованный хаб для конфигурации сервисов. Подгружает конфиги из [aston-configRepo](https://github.com/gedfalk/aston-configRepo
)

Запуск через [aston-serviceOrchestration](https://github.com/gedfalk/aston-serviceOrchestration)
